How EggCoin Works

EggCoin Finance introduces a decentralized stablecoin model where each `(EGG$ + CHI)` pair is guaranteed by `$2.00` in locked crypto assets.

Built on immutable ERC-20 tokens:

üü¢ EGG$ ‚Äì Soft-Pegged Stablecoin

  • No risk ‚Äì nominal value only
  • Redeemable only with full pair

üü° CHI ‚Äì Surplus/Risk Token

  • In crisis, `CHI` devalues before collateral

  7. `CHI` holders get proportional rewards
  8. The system self-stabilizes through user incentives
